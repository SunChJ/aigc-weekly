# Daily content schema

Each daily issue is a Markdown file under `content/daily/`.

Filename (recommended): `YYYY-MM-DD.md`

Frontmatter fields:

```yaml
---
slug: 2026-02-04
# NOTE: these are examples; quotes may be required depending on your YAML parser.
title: AI 时代·日刊精选
publishDate: 2026-02-04
summary: ''
tags:
  - ai
  - tools
status: published # published | draft
---
```

Body: Markdown content.
